## 🚀 Quick Start

### Option 1: Automatic Setup
```bash
# Double-click start-local.bat
start-local.bat
```

### Option 2: Manual Setup
```bash
# Backend
cd backend
npm install
npm run seed
npm run dev

# Frontend (new terminal)
cd frontend
npm install
npm run dev
```

## 🌐 Access Points
